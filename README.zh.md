# 学校杀

[English](./README.md) | **Chinese**

学校杀是由Stoller学生共同创造的类三国杀卡牌游戏
> 完整名称： Stoller SchoolSha | 思多乐学校杀

## 制作人员
> 排名不分先后

- 角色与策划：孽慈，棉兔砸，[皮蛋](https://github.com/bretren)
- 审核：孽慈，Tim

## 起源
某天在玩三国杀的时候突然想做一个学校版本的然后几个人就可以做了。

然后因为暑假和一些原因导致纸质版的成功的废弃了。
