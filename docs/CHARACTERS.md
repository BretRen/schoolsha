# 角色与技能

学校杀目前有 4 个可选角色，每个角色有独特的技能和体力值。

---

## 角色一览

| 角色         | ID        | 体力 | 技能         | 定位                     |
| ------------ | --------- | ---- | ------------ | ------------------------ |
| **普通学生** | `normal`  | 3    | 无           | 白板，适合新手           |
| **班长**     | `monitor` | 3    | 权威         | 控制型，每回合可弃对手牌 |
| **体育生**   | `athlete` | 4    | 强壮（锁定） | 血牛，手牌上限高         |
| **学霸**     | `nerd`    | 3    | 补习（锁定） | 过牌型，每回合多摸一张   |

---

## 技能详情

### 权威（班长）

- **类型**：主动技
- **限制**：每回合限 1 次
- **阶段**：出牌阶段
- **消耗**：弃 1 张手牌
- **效果**：对手随机弃 1 张手牌

> _"交作业！"——班长一声令下，谁敢不从。_

### 强壮（体育生）

- **类型**：锁定技
- **效果**：手牌上限 +1

> 体育生 4 点体力，加上强壮效果，手牌上限 = 当前体力 + 1，续航能力极强。

### 补习（学霸）

- **类型**：锁定技
- **触发**：摸牌阶段
- **效果**：额外摸 1 张牌（每回合共摸 3 张）

> 别人摸 2 张，学霸摸 3 张。知识就是力量。

---

## 技能系统说明

### 技能类型

| 类型                 | 说明                                  |
| -------------------- | ------------------------------------- |
| **主动技 (active)**  | 玩家手动触发，可设定 perTurn 次数限制 |
| **锁定技 (locked)**  | 自动生效，不可关闭，触发时自动执行    |
| **被动技 (passive)** | 事件触发自动执行（与锁定技实现相同）  |

### 技能架构

技能采用 JSON 驱动设计，新增一个技能只需：

1. 在 `skills.json` 中添加技能定义（id、name、type、trigger、effect）
2. 如需新的效果类型，在 `skills.ts` 的 `executeSkillEffect` 中添加对应的 case

技能通过事件总线 (`events.ts`)
绑定到游戏中的各种事件（摸牌、受伤、回合开始等），无需修改游戏核心逻辑。
